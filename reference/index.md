# Package index

## All functions

- [`bio_to_math()`](https://xsdm-project.github.io/xsdm-devel/reference/bio_to_math.md)
  : Converts parameters from the biological scale to the math
  (unconstrained) scale

- [`build_orthogonal_matrix()`](https://xsdm-project.github.io/xsdm-devel/reference/build_orthogonal_matrix.md)
  : Build an orthogonal matrix from a real-parameter vector

- [`convert_equivalence_class()`](https://xsdm-project.github.io/xsdm-devel/reference/convert_equivalence_class.md)
  : Converts a set of parameters to other representatives of the same
  equivalence class

- [`create_mask()`](https://xsdm-project.github.io/xsdm-devel/reference/create_mask.md)
  : Create a parameter mask aligned with \`make_mask_names()\`

- [`create_param_vector_masked()`](https://xsdm-project.github.io/xsdm-devel/reference/create_param_vector_masked.md)
  : Create a complete parameter vector with canonical names (no NAs
  allowed)

- [`dist_between_params()`](https://xsdm-project.github.io/xsdm-devel/reference/dist_between_params.md)
  : Distance in parameter space between two sets of parameters

- [`env_data_array()`](https://xsdm-project.github.io/xsdm-devel/reference/env_data_array.md)
  : Get an array of environmental data from presence-absence points.

- [`example_1`](https://xsdm-project.github.io/xsdm-devel/reference/example_1.md)
  : Consolidated example data for the xsdm package

- [`example_2`](https://xsdm-project.github.io/xsdm-devel/reference/example_2.md)
  : Consolidated example data for the xsdm. This is environmental data
  array and an occurrence presence absence vector of Ophisaurus
  ventralis. A named list containing all example datasets used in the
  package's documentation and examples.

- [`example_3`](https://xsdm-project.github.io/xsdm-devel/reference/example_3.md)
  : Consolidated example data for the xsdm. This is environmental data
  array and an occurrence presence absence vector Blarina carolinensis A
  named list containing all example datasets used in the package's
  documentation and examples.

- [`expit()`](https://xsdm-project.github.io/xsdm-devel/reference/expit.md)

  exp(x))

- [`extract_orthogonal_matrix_parameters()`](https://xsdm-project.github.io/xsdm-devel/reference/extract_orthogonal_matrix_parameters.md)
  : Extract a math-scale real-parameter vector corresponding to a
  special orthogonal matrix

- [`habitat_suitability()`](https://xsdm-project.github.io/xsdm-devel/reference/habitat_suitability.md)
  : Tiled habitat-suitability map from environmental raster stacks

- [`interpret_parameters()`](https://xsdm-project.github.io/xsdm-devel/reference/interpret_parameters.md)
  : Tool to help interpret xsdm model parameters

- [`like_ltsg`](https://xsdm-project.github.io/xsdm-devel/reference/like_ltsg.md)
  : Compute likelihood for LTSG model

- [`like_neg_ltsgr()`](https://xsdm-project.github.io/xsdm-devel/reference/like_neg_ltsgr.md)
  : Long-term stochastic growth rate worker function for the xsdm model

- [`log1mexp()`](https://xsdm-project.github.io/xsdm-devel/reference/log1mexp.md)
  : Numerically stable \`log(1 - exp(-a))\`

- [`log1pexp()`](https://xsdm-project.github.io/xsdm-devel/reference/log1pexp.md)
  : Numerically stable \`log(1 + exp(x))\`

- [`log_prob_detect()`](https://xsdm-project.github.io/xsdm-devel/reference/log_prob_detect.md)
  : Probability of detection of the species in each location

- [`log_prob_detect_cpp()`](https://xsdm-project.github.io/xsdm-devel/reference/log_prob_detect_cpp.md)
  : Compute log detection probabilities from a flat environmental data
